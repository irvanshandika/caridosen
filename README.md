# CariDosen

CariDosen merupakan website untuk mencari dosen serta memberikan rating performa mengajar dari sang dosen.
## Fitur

## Untuk superadmin
- Manajamen User
- Tambah, Edit, dan Delete Dosen
## Untuk pengguna non admin
- Memberikan rating serta komentar
- Update Profil
- Delete Profil
- Support PWA
